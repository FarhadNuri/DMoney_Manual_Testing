# Dmoney Manual Testing

## Project Description

This repository presents a professional-grade manual testing project for the **Dmoney REST API**, focusing on financial transaction operations like deposits, transfers, withdrawals, and balance inquiries. It includes a complete suite of manual test cases, detailed bug reports, completion reports, and documentation to demonstrate best practices in quality assurance.

## Key Features
* Comprehensive manual testing covering functional, boundary, and negative test scenarios
* End-to-end manual test execution verifying actual behavior against expected outcomes
* Detailed completion reporting with execution insights and overall pass/fail metrics
* Validation of API responses, status codes, authentication, and business logic
* Well-documented bug reports and improvement recommendations for better software quality
* Structured traceability and systematic QA checklists
